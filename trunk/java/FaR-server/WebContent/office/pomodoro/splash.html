• 番茄工作法(The porodoro technique)是一种简单易行的时间管理方法,它的核心思想是集中精力完成工作,排除干扰,合理安排休息时间。
  • 为了实现这一目的,番茄时间的概念被提了出来,一个番茄时间包括25分钟的工作和5分钟的休息,番茄时间是不能被分割或者延长的,你必须严格遵守它,并且通过记录形式来查看您的执行程度。
  • 番茄工作法这款软件就是为了辅助您完成这些工作的,它涵盖了记录番茄的所有方面:
    • 制订计划
    • 记录工作时间和休息时间
    • 记录中断，并把中断记录到计划中
    • 记录未来事务，并把事务加入计划中
    • 计划的执行情况查看
    • 番茄的统计
