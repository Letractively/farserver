G网盘 name:webdisk

Yaomy

G网盘的目的就是让您像操作本地磁盘文件一样操作自己的web服务器上的文件。在大家都拥有多个终端设备的今天，用不同设备能随时存取相同的文件夹显得很关键。 您可以在自己购买甚至自己安装的Far服务器上直接安装这个应用，G网盘将来还会提供各种同步客户端供试用。

即时时间管理 name:mytime

Yaomy

时间就是生命。即时时间管理可以让您更好的跟踪您的每日时间分配情况，当您开始或结束某项工作时都去点击一下应用，这样虽然会给您带来额外的操作，但是每天下来您就很清楚在哪个事情上花掉了多长时间，以敦促您更好的使用时间。 另外如果您在服务器上设置了goole帐号，您也可以把每天的日程同步到google日历上
