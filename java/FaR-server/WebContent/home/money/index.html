欢迎使用进度计划小助手

进度计划小助手是一款简单好用的项目管理工具。
相比像控制复杂与严格的project等软件来说，进度计划小助手更侧重于对项目任务的分解、任务的进展、资源、附加文件、即时进展等方面功能。
使用者首先针对将工作或事务建立项目，然后在项目中增加子任务。 而针对于子任务， 可以设置起止时间、完成进度、资源、紧急程度等信息。
另外，也可以进行添加任务进展的备注消息、附加文件等操作。
